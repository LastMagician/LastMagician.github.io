关于Shadowsocks的一些技巧

Apr 2, 2016

众所周知，我们使用Shadowsocks是为了绕过 GFW 的封锁，去上一些我们想上却又上不了网站，

解决一些网站无法访问的问题

有童鞋会发现，使用 Shadowsocks 之后虽然大部分被墙的网站我们都可以上，比如google、 facebook等等一些网站，但还是有一些网站上不了，比如 Android Developers 的网站，如果我们是自己搭建的 ss 服务，我们首先会测试一下网站能不能ping通，当然一般是可以通的。其实这里就涉及到本地Shadowsocks的系统代理模式的问题，它一般有两种模式，一种是PAC模式，一种是全局模式。这里有两种解决方法，如下：

  • 第一种方法，改用全局模式，启用之后你会发现原先无法打开的网站能打开了(如：Android Developers)。
  • 第二种方法，编辑PAC文件，
    • 右击小飞机的图标，选择编辑本地PAC文件，一般是pac.txt的文件；
    • 打开后会发现有类似下面的代码：
var domains = {
  "gimpshop.com": 1, 
  "directcreative.com": 1,

这时我们只要按照相应的格式将无法访问的网站链接添加进去就好了，比如 Android Developers 的网站 developer.android.com，保存之后重新打开你会发现可以访问了！

var domains = {
  "developer.android.com": 1, 
  "gimpshop.com": 1, 
  "directcreative.com": 1,
