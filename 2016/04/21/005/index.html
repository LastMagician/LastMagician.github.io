关于使用github过程中会遇到的一些问题

Apr 21, 2016

1.为什么git总是要求输入我的密码？

在Window下，当我们使用HTTPS URL的方式 clone 时，我们每次与github交互时都会提示输入用户名和密码。

使用HTTPS的远程URL有一定的优势，它比起SSH更容易设置，通常可以在严格的防火墙和代理下工作。然而，当我们使用pull和push时每次都会提示输入账号。

我们可以配置Git存储我们的密码。命令如下：

$ git config --global credential.helper wincred
